# Image Prompts: Method Name

These prompts are support material for contributors.
They are conceptual only and do not create final project evidence.

## Concept visual prompt

### Purpose

Create a conceptual only teaching visual that helps users understand the method idea.

### Use

training and explanation only.
not final project evidence.
not a quantitative chart.

### Prompt

Create a clean professional training visual that illustrates the method concept without exact numerical values.
Use simple shapes, generic process context, and enough open space for the Markdown guide to carry detailed explanation.

### Negative constraints

Do not include exact data values.
Do not include fake percentages.
Do not include misleading axes.
Do not imply the image is project evidence.
Do not include private operational data, customer names, employee names, supplier names, patient data, credentials, or hidden sensitive notes.
Keep detailed method instructions in Markdown rather than rendering them as small image text.

## Good/bad comparison prompt

### Purpose

Create a conceptual only teaching visual that compares a good layout with a bad layout.

### Use

training and explanation only.
not final project evidence.
not a quantitative chart.

### Prompt

Create a clean professional training visual comparing an acceptable method output layout with a weak layout.
Use generic shapes and visual hierarchy only.
Avoid precise values, fake percentages, detailed axes, and small instructional text.

### Negative constraints

Do not include exact data values.
Do not include fake percentages.
Do not include misleading axes.
Do not imply either side is project evidence.
Keep detailed method instructions in Markdown.

## Review checklist

| Check | Pass condition |
|---|---|
| Conceptual only | The image cannot be confused with final project evidence. |
| No exact numeric claims | The image avoids exact values, fake percentages, and precise axes. |
| Text-light | Detailed wording remains in Markdown. |
| Method-correct | The visual does not teach the wrong method idea. |
| Professional quality | The visual is suitable for a QCC training guide. |
