# Image Prompt: Pareto Good/Bad Comparison

## Purpose

Create a conceptual only teaching visual comparing a good Pareto chart layout with a bad Pareto chart layout.

## Use

training and explanation only.
not final project evidence.
not a quantitative chart.

## Prompt

Create a clean professional training visual with two simplified panels.
The good panel should suggest descending bars, readable labels, and a source-note area.
The bad panel should suggest unsorted bars, clutter, and missing evidence context.
Use no exact numerical values.
Keep detailed method instructions in Markdown.

## Negative constraints

Do not include exact data values.
Do not include fake percentages.
Do not include misleading axes.
Do not imply the image is project evidence.
Do not render detailed review instructions inside the image.
Keep detailed method instructions in Markdown.

## Review checklist

| Check | Pass condition |
|---|---|
| Conceptual only | The visual teaches layout quality but is not final evidence. |
| No exact numeric claims | The visual avoids exact values and fake percentages. |
| Text-light | Detailed explanation remains in Markdown. |
| Good/bad distinction | The contrast is recognizable without tiny text. |
