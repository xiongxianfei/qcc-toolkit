# Image Prompt: Pareto Concept Visual

## Purpose

Create a conceptual only teaching visual for the Pareto Chart method.

## Use

training and explanation only.
not final project evidence.
not a quantitative chart.

## Prompt

Create a clean professional training visual showing the idea of vital few versus many minor categories in quality defects.
Use generic bars without exact numerical values.
Keep the visual text-light and leave detailed method instructions in Markdown.

## Negative constraints

Do not include exact data values.
Do not include fake percentages.
Do not include misleading axes.
Do not imply the image is project evidence.
Keep detailed method instructions in Markdown.

## Review checklist

| Check | Pass condition |
|---|---|
| Conceptual only | The image cannot be confused with a final project chart. |
| No exact numeric claims | The image avoids exact values, fake percentages, and precise axes. |
| Text-light | Detailed wording remains in Markdown. |
| Method-correct | The visual shows concentration without claiming root cause. |
