# Teaching Visuals

Reviewed teaching visuals for the Pareto Chart method belong here.

## Current status

No binary teaching visuals are included in M2.
The reviewed good and bad examples are recorded as Markdown teaching notes in `examples/`.

## Use boundary

Teaching visuals are conceptual only.
They are not final project evidence and must not include exact project data values, fake percentages, misleading axes, or private operational data.
