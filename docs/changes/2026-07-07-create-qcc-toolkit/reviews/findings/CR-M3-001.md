- Safe resolution path: Use Plotly's self-contained HTML mode, such as `include_plotlyjs=True`, or another local/offline rendering approach. Add a test that fails if `chart.html` contains a Plotly CDN URL or `include_plotlyjs="cdn"` style output. Rerun M3 validation after the fix.
