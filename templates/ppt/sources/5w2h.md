## Presentation pattern

Use these source notes as the reviewable contract for the generated PowerPoint template.
The deck includes a data-entry slide with editable 5W2H answer cells.
The final slide should turn the broad issue into a specific, measurable problem statement.

## Method kit sections

### Purpose

Turn a broad concern into a specific, reviewable QCC problem statement.

### QCC stage fit

Define Problem. Use before detailed data collection and target setting.

### When to use

Use when the problem statement is vague, too broad, or missing measurable context.

### When not to use

Do not use it as root-cause proof or as a place to hide the proposed countermeasure.

### Required inputs

Observed problem background, what/why/where/when/who/how/how much answers, source, date range, known assumptions, and measurable impact.

### Completed demo example

The deck includes a completed DEMO EXAMPLE - not project evidence that shows realistic 5W2H answers.

### Blank working slide or worksheet

The deck includes a blank working slide or worksheet users can copy into a QCC project deck.

### Interpretation patterns

Use patterns for problem statement, scope decision, unknown fact gap, and caution wording that the slide does not prove root cause.

### Common mistakes

Avoid opinions instead of observations, solution wording inside the problem, missing measurable impact, missing source or date range, and demo content used as project evidence.

### Facilitator checklist

Check that each W/H answer is specific, source and date range are shown, the statement avoids countermeasures, the key conclusion is written, and the next action is clear.

### Python assist decision

PowerPoint is enough for teaching, draft, and normal 5W2H use.
Python is not normally needed unless impact numbers require large-data summary or high-rigor reproducible evidence.

### Evidence/source note

Record source, date range, assumptions, prepared-by/date, and whether evidence level is teaching/draft, normal QCC project, competition or management review, or audit/high-risk.
