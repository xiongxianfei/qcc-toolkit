# Incoming Templates

This folder is for user-created or legacy source templates.
Files here are not official method kits and must not be treated as reviewed QCC Toolkit assets.

## Rules

- Incoming templates are source assets only.
- Do not add an incoming file to the official catalog entry set until it passes the method-kit quality standard.
- Record the template owner, method, QCC stage, and review status when known.
- If the asset has unknown method, unknown QCC stage, unknown owner, or unsupported formulas, it remains source material.
- Do not publish private data from this folder.

## Required Review

Before migration into an official method kit, check for:

- real customer names;
- employee names;
- supplier names;
- patient data;
- credentials;
- hidden notes;
- unsupported formulas;
- private data;
- stale examples;
- undocumented assumptions.

Useful layouts or wording can be migrated into an official method kit only after review.
The official kit must have a Markdown guide, PowerPoint source notes, PowerPoint template, demo label, blank working surface, facilitator checklist, Evidence/source note, Python assist decision, and catalog entry.
